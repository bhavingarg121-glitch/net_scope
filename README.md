
📡 NetScope India

🚀 Real-Time Telecom Intelligence & Network Comparison Platform for India

NetScope is a map-based AI-powered telecom analytics system that visualizes network performance (Jio, Airtel, Vi, BSNL), simulates signal strength, and predicts network quality using real-time location data and AI logic.
